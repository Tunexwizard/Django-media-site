# 5th Contact Session - Django Media Upload App

## Instructions to Run:

1. Open the VSCode terminal in this folder
2. Activate virtual environment:
   - `venv\Scripts\activate`
3. Run the server:
   - `python manage.py runserver`
4. Open in browser:
   - `http://127.0.0.1:8000/`

## Features:
- Upload and display media files (e.g., MP4, images)
- Files stored in media, I uploaded an anime image to test
